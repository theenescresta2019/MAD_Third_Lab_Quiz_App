# MAD_Third_Lab_Quiz_App
## Quiz Time

This app has three activity, Main Activity, Quiz Activity and Score Activity. This app consists of multiple questions which carry 10 marks each. In the case of multiple answers, the points will be divided per answer. Marks will be shown after clicking the submit button.

Intent Method is used to communicate with the multiple layouts. An Intent is a simple message object that is used to communicate between android components such as activities, content providers, broadcast receivers and services. Intents are also used to transfer data between activities. Intents are used generally for starting a new activity using startActivity().

###  The final score and message is shown using Intent Method.
